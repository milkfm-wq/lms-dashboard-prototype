# Feedback Files

Складывайте сюда файлы, которые экспортирует наставник из прототипа:

- `dashboard-feedback-vXX-YYYY-MM-DD.json`
- `dashboard-feedback-vXX-YYYY-MM-DD.md`

Пример:
- `dashboard-feedback-v01-2026-03-30.json`
- `dashboard-feedback-v01-2026-03-30.md`

После загрузки новых файлов:
- обновите `UX Feedback Log.md`;
- зафиксируйте решения в `Dashboard Spec v1.md`;
- выпустите следующую итерацию прототипа (`v02`, `v03`, ...).
