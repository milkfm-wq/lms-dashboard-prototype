# Role Visibility Matrix

Матрица видимости по ключевым блокам дашборда.

| Block ID | Блок | Mentor | Assistant | Student | Комментарий |
|---|---|---|---|---|---|
| M1 | Общий прогресс группы | Full | Read | Limited/None | уточнить в MVP |
| M2 | Средний рейтинг | Full | Read | Limited | уточнить |
| M3 | В зоне риска | Full | Read | Limited | без внутренних комментариев |
| C1 | Динамика группы | Full | Read | Limited | зависит от режима видимости |
| T1 | Таблица участников | Full | Read | Own/Limited | student без чужих деталей |
| S1 | Карточка ученика | Full | Read | Own only | строго own-only для student |

## Статусы доступа

- `Full`: чтение + управленческие действия.
- `Read`: только просмотр.
- `Own only`: только собственные данные.
- `Limited`: ограниченный набор полей.

## Примечание

Финальная матрица должна совпадать с правилами из auth/middleware и API.
